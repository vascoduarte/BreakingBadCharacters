# BreakingBadCharacters
 BreakingBadCharacters is a Simple application developed in Kotlin that uses the The Breaking Bad API(https://breakingbadapi.com/) 
 
Using MVVM design pattern and AndroidX libraries:

● ViewModels

● LiveData

● NavigationUI

● Room

● Kotlin Coroutines

● Retrofit

● OkHttp

● Glide

● Hilt + Dagger 2

● Espresso
